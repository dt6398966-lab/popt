# Images Directory

Add the following images to this folder:

## Required Images:

1. **99acres-logo.png** (or **logo-placeholder.png**)
   - Header logo
   - Recommended size: 80px height (width auto)
   - Format: PNG with transparency

## Optional Images:

2. **free-badge.png**
   - "Free" badge for Post Property button
   - Recommended size: 32px x 16px
   - Format: PNG

3. **banner-ad.jpg**
   - Top banner advertisement
   - Recommended size: 1366px width x 340px height
   - Format: JPG/PNG
   - To enable: Remove `style="display: none;"` from banner div in header.ejs

## Image Guidelines:

- Use PNG for logos and badges (supports transparency)
- Use JPG for photos/banners (smaller file size)
- Optimize images before uploading for better performance
- Keep file sizes under 200KB when possible

## Placeholder:

If images are missing, the code will:
- Hide the banner automatically
- Hide the "Free" badge if not found
- Use placeholder logo if main logo not found
